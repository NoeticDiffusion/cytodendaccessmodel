cytodendaccessmodel documentation
=================================

Companion code for *A Cytoskeletal-Dendritic Accessibility Model of
Associative Memory*.

The repository contains a branch-resolved associative-memory simulator, DANDI
open-data tooling, reviewer reproduction guides, and manuscript-facing
experiment entry points.

.. important::

   The distribution name is ``asomemm`` for legacy compatibility, while the
   import packages are ``cytodend_accessmodel``, ``dandi_analysis``, and
   ``dandi_io``.
