# Output Manifest

This file lists the reviewer-facing artifacts produced by the experiment
commands in `RUN.md`. Paths are relative to the repository root.

## Simulator Outputs

Most `experiments/exp*.py` scripts print their primary tables and verdicts to
the terminal. The canonical paper summary additionally writes:

- `data/reviewer/013_canonical_values.json` from
  `python experiments/exp013_paper_summary.py`

Executable-article SVG figures are generated by:

```bash
python experiments/gen_figures_executable.py
```

Output directory:

- `.article/Executable Structural Accessibility - A Biologically Constrained Cytoskeletal-Dendritic Model of Memory Linking/figures/`

Expected files:

- `figure_E1_architecture.svg`

- `figure_E4_three_factor.svg`
- `figure_E5_ablation.svg`
- `figure_E6_comparator_heatmap.svg`
- `figure_E7_motif_family.svg`
